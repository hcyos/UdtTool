# UDT Tool

这是一个用于开源小工具，可以把从博途导出的DB源文件转换为汇川触摸屏能识别的csv文件。如果你有这方面的需求，可以尝试使用这个工具。

## 使用方法
1. 从博途导出DB源文件，文件格式为`.db`，需要把所有关联块都导出且DB块是非优化的。
2. 使用本工具打开`.db`文件，选择需要导出的DB块，右键选择`导出`，在选项窗口中输入导出的文件路径，DB块的编号，PLC的连接名称，点击`确定`按钮。
3. 等待导出完成，导出的文件会保存在指定的路径下。
